## What MCP is

The **Model Context Protocol** is an open specification for connecting AI clients (like Claude Desktop, Cursor, Claude Code) to external tools and data sources. It is published at [modelcontextprotocol.io](https://modelcontextprotocol.io) and developed in the open on [GitHub](https://github.com/modelcontextprotocol).

In a WordPress context, the protocol is implemented by the official [WordPress MCP Adapter](https://github.com/WordPress/mcp-adapter) plugin. The Adapter exposes a WordPress site's abilities (registered via the Abilities API) to MCP-compatible clients.

Inhale: MCP Abilities sits on top of the MCP Adapter and provides a settings UI for choosing which abilities the Adapter exposes.

## the Inhale: MCP Abilities plugin's relationship to Anthropic

MCP was **originally developed by Anthropic** as part of their work on Claude. Anthropic has since opened the specification for community contribution, and the protocol is now implemented by multiple AI clients beyond Anthropic's own products.

**Inhale: MCP Abilities is a third-party plugin.** It is not affiliated with, endorsed by, or sponsored by Anthropic. Respira, the company that maintains Inhale: MCP Abilities, is an independent company with no business or organizational relationship to Anthropic.

Anthropic gets mentioned Anthropic in the docs and the plugin description because:

1. They created the specification Inhale: MCP Abilities depends on, and crediting them is correct.
2. The most common MCP client our users will connect (Claude Desktop) is an Anthropic product.

Neither of those facts implies any partnership.

## About Respira

Respira is an independent company that ships AI infrastructure for WordPress. Inhale: MCP Abilities is offered free; Respira's commercial work centers on **Respira for WordPress**, a paid safety layer for AI-driven edits across 12 page builders.

Inhale: MCP Abilities and Respira for WordPress are technically and commercially separate. Inhale: MCP Abilities will continue to work whether or not you ever install Respira for WordPress.
