### What's New in Respira for WordPress 7.5.0

The release where connecting your AI and editing your site finally tell the same story.

**Versions**
- Plugin: 7.5.0
- MCP server: `@respira/wordpress-mcp-server` 7.2.4

**Connection, rebuilt**
- One place your connection state lives, shared by the plugin, the dashboard, and your agent.
- The setup and connection screens were rebuilt around live state, so wp-admin and respira.press tell the same story.
- One-click installs lead the way in: Claude Desktop, Cursor, and Claude Cowork. Browser sign-in is still there for ChatGPT and claude.ai.
- Name your sites: give a site a friendly name so a long staging URL becomes something you can say out loud, and your agent sees the name too.

**Builder fixes on real installs**
- Bricks: moving or nesting an element keeps the elements next to it, and a link set to open in the same tab now does.
- Divi: single-module edits refresh the cache so the page is not left stale, and bracketed color values no longer corrupt a custom post type.
- Elementor 4.x and Gutenberg: fixes verified against live installs.

For the full notes, see the [7.5.0 release page](https://www.respira.press/releases/7.5.0). All releases are listed at [respira.press/releases](https://www.respira.press/releases).

</Update>
